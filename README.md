旅行が好きで好きでたまらないので、まずはこれを作ってみました！

航空券検索UI（往復対応）  
Next.js × TypeScript を使って構築した入門向けポートフォリオプロジェクトです。

## 🎯 機能一覧

- ✅ 出発地／到着地の選択
- ✅ 往路・復路の日付入力（復路は往路より後の日付のみ選択可能）
- ✅ 各航空会社（ANA, JAL, Peach, Jetstar）からのデータ統合（jsonで管理)
- ✅ 曜日に応じた価格倍率（例：金曜は高め、日曜は安め）
- ✅ 検索結果をカード形式で表示
- ✅ フライト時間（hour/min）を表示
- ✅ 合計金額表示
